A simple PowerShell script that provides a graphical interface (GUI) to manage Hyper-V Extended ACL (firewall) rules on Windows Server 2016 and later.
    Manage Rules: Easily list, add, edit, and delete firewall rules.
    Scope: Supports both Virtual Machine network adapters and the Management OS.
    Requirement: The script must be run with Administrator privileges.


Editor GUI para Hyper-V Extended ACL

Un script de PowerShell sencillo que proporciona una interfaz gráfica (GUI) para gestionar las reglas ACL extendidas (firewall) de Hyper-V en Windows Server 2016 y versiones posteriores.
    Gestión de reglas: Permite listar, añadir, editar y eliminar reglas de firewall fácilmente.
    Alcance: Soporte para adaptadores de red de Máquinas Virtuales y el SO de administración.
    Requisitos: El script debe ejecutarse con privilegios de Administrador.
