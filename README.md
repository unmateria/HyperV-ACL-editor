# HyperV-ACL-editor
a simple powershell GUI editor for Hyper-V Extended ACL (firewall) rules on Windows Server 2016 and later

This PowerShell script provides a graphical interface to manage Hyper-V extended ACL rules.
It allows you to list, add, edit and delete rules on virtual machine network adapters or the management OS.
The script must be run with administrator privileges.
